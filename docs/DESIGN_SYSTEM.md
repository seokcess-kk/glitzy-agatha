## 2. 타이포그래피

### 폰트

| 용도 | 폰트 | 비고 |
|------|------|------|
| 본문/제목/라벨 | **Pretendard** | 한글 최적화, Inter 기반 |
| KPI 숫자 | **Geist Mono** | 숫자 가독성 특화 |

### Weight

| Weight | 용도 |
|--------|------|
| 400 (Regular) | 본문, 설명 텍스트 |
| 500 (Medium) | 라벨, 테이블 헤더 |
| 600 (SemiBold) | 제목, KPI 숫자 |
| 700 (Bold) | 강조 |

### KPI 카드 예시

```
┌─────────────┐
│ 리드          ← Pretendard 500, 작은 사이즈, Slate-500
│ 150건        ← Geist Mono 600, 큰 사이즈, Slate-900
│ ▲ 12%       ← Geist Mono, Emerald
└─────────────┘
```

## 3. 레이아웃

### 사이드바

- **토글 방식**: 64px (축소) ↔ 240px (확장), 버튼 토글
- 상태 localStorage 저장
- 모바일: 완전 숨김 + 햄버거 → 오버레이 확장

**축소 상태 (64px):**
- 로고 심볼만 표시
- 메뉴 아이콘만, 호버 시 tooltip
- 클라이언트 스위처: 아이콘 클릭 → popover

**확장 상태 (240px):**
- 로고 심볼 + 워드마크 + 토글 버튼 `[◀]`
- 클라이언트 스위처 드롭다운
- 메뉴 아이콘 + 텍스트 라벨

### 사이드바 메뉴 구성

**일반 메뉴 (전체 역할):**

| 그룹 | 항목 | 아이콘 (Lucide) | 구분 |
|------|------|----------------|------|
| — | 대시보드 | `LayoutDashboard` | |
| — | 광고 성과 | `BarChart3` | 얇은 구분선 |
| — | 캠페인 | `Megaphone` | |
| — | 고객관리 | `Users` | |
| — | 순위 모니터링 | `TrendingUp` | 얇은 구분선 |

**관리 메뉴 (superadmin 전용):**

| 그룹 | 항목 | 아이콘 (Lucide) | 구분 |
|------|------|----------------|------|
| 관리 | 클라이언트 | `Building2` | 라벨 "관리" |
| 관리 | 계정 관리 | `UserCog` | |
| 관리 | 광고 소재 | `Image` | |
| 관리 | 랜딩페이지 | `FileText` | |
| 관리 | UTM 관리 | `Link2` | |
| 관리 | 설정 | `Settings` | |

**아이콘 색상:**
- 기본: `Slate-400`
- 활성 (현재 페이지): `Violet-600`
- 호버: `Slate-400` → `Violet-500`

**활성 메뉴 스타일:**
- 아이콘 + 텍스트: `Violet-600`
- 배경: `Violet-600/5`
- 왼쪽 2px 인디케이터 바: `Violet-600`

### 반응형 브레이크포인트

Tailwind CSS 기본 브레이크포인트 사용:

| 이름 | 크기 | 용도 |
|------|------|------|
| sm | 640px | 모바일 |
| md | 768px | 태블릿 (사이드바 전환점) |
| lg | 1024px | 소형 데스크탑 |
| xl | 1280px | 데스크탑 (KPI 4열 전환점) |

### 콘텐츠 영역

| 디바이스 | 페이지 패딩 |
|----------|-----------|
| 데스크탑 | 32px (p-8) |

### 간격

| 요소 | 간격 |
|------|------|
| KPI 카드 간 | 16px (gap-4) |
| 섹션 간 | 24px (gap-6) |

| 디바이스 | 열 수 |

## 4. 컴포넌트

### 카드

| 요소 | 값 |
|------|-----|
| 배경 | white (라이트), Slate-900 (다크) |
| 보더 | 1px `Slate-200` (라이트), `Slate-800` (다크) |
| 섀도우 | 없음 |
| Radius | 8px |
| 패딩 | 20px |
| 호버 | 보더 → `Violet-600` |

### 버튼

| 유형 | 배경 | 텍스트 | 호버 |
|------|------|--------|------|
| Primary | `Violet-600` | white | `Violet-700` |
| Secondary | `Slate-100` | `Slate-700` | `Slate-200` |
| Ghost | 투명 | `Slate-700` | `Slate-100` |
| Destructive | `Rose-600` | white | `Rose-700` |

- Radius: 8px
- 패딩: 10px 16px
- Font: Pretendard 500

### 테이블

| 요소 | 스타일 |
|------|--------|
| 헤더 | `Slate-50` 배경, `Slate-500` 텍스트, Pretendard 500 |
| 행 구분 | 하단 `Slate-100` 보더 1px |
| 행 호버 | `Slate-50` 배경 |
| 선택된 행 | `Violet-50` 배경, 왼쪽 2px `Violet-600` 바 |
| 셀 패딩 | 12px 세로, 16px 가로 |
| 정렬 | 첫 열(이름) 왼쪽, 나머지 숫자 열 오른쪽 |
| 숫자 폰트 | Geist Mono |
| 정렬 아이콘 | `Slate-400`, 활성 시 `Violet-600` |
| 빈 상태 | 중앙 아이콘 + "데이터가 없습니다" |

### 모달/다이얼로그

- 기반: shadcn/ui Dialog (Radix UI)
- 오버레이: `black/40`, 블러 없음
- 배경: white, 보더 `Slate-200`
- Radius: 8px
- 패딩: 24px
- 최대 폭: sm(400px), md(500px), lg(640px)
- 제목: Pretendard 600, 18px
- 닫기: 우상단 X 아이콘 `Slate-400`
- 하단 버튼: 오른쪽 정렬, 취소(Ghost) + 확인(Primary)
- 애니메이션: fade + scale(0.95→1), 150ms

### 토스트

- 라이브러리: Sonner
- 위치: **상단 중앙** (top-center)
- 배경: white, 보더 `Slate-200`, radius 8px
- 너비: 360px
- 유형별 왼쪽 바: 성공 Emerald, 에러 Rose, 경고 Amber, 정보 Violet
- 자동 닫힘: 4초
- 닫기: X 버튼 + 스와이프

### 아이콘

- 라이브러리: Lucide React
- 기본 색상: `Slate-400`
- 활성/강조: `Violet-600`
- 사이즈: 메뉴 17px, 일반 16px, 소형 14px

### 상태 배지

리드 상태별 배지 색상:

| 상태 | 색상 | 코드 | 스타일 |
|------|------|------|--------|
| 신규 (new) | Violet | `#7C3AED` | 배경 Violet-50, 텍스트 Violet-700 |
| 진행중 (in_progress) | Amber | `#D97706` | 배경 Amber-50, 텍스트 Amber-700 |
| 전환 (converted) | Emerald | `#059669` | 배경 Emerald-50, 텍스트 Emerald-700 |
| 보류 (hold) | Slate | `#64748B` | 배경 Slate-100, 텍스트 Slate-600 |
| 미전환 (lost) | Rose | `#E11D48` | 배경 Rose-50, 텍스트 Rose-700 |
| 무효 (invalid) | Slate-light | `#CBD5E1` | 배경 Slate-50, 텍스트 Slate-400 |

- Radius: 4px (카드보다 작게)
- 패딩: 2px 8px
- Font: Pretendard 500, 12px

### 폼 컴포넌트

| 요소 | 스타일 |
|------|--------|

## 6. 디자인 원칙

| 원칙 | 설명 |
|------|------|
| **미니멀** | 불필요한 장식 제거 — 글로우, 그라디언트 배경, 네온 효과 없음 |
| **여백** | 빡빡한 배치 대신 넉넉한 spacing으로 호흡 |
| **타이포 중심** | 큰 숫자, 명확한 위계. 아이콘보다 텍스트가 정보 전달 |
| **데이터 강조** | 배경은 뒤로, 숫자와 차트가 주인공 |
| **미세 디테일** | 호버 시 보더 Violet 전환, 얇은 인디케이터 바 등 절제된 인터랙션 |
| **참고 톤** | Vercel / Linear 라이트 버전 |
